How It Works

The new stack behind ingestion, storage, and visualization.

  • SQLite stores ticker history, news articles, and ingest run metadata locally.
  • Alpha Vantage is the first live provider, with bundled sample data as a fallback seed.
  • The server exposes JSON endpoints for bootstrap, market data, news data, and ingest.
  • Three.js renders a rotating 3D bar field for the selected ticker.
  • TACO signal scoring still combines dip severity, rebound, and reversal evidence.
